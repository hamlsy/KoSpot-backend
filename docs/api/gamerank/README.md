# GameRank API Documentation

게임 랭크 관련 API 문서입니다.

---

## 레이팅 점수 계산 테스트
**GET** `/gameRank/{currentRatingPoint}/{gameScore}`

현재 레이팅 점수와 게임 점수를 기반으로 레이팅 변화량을 계산하는 테스트 API입니다.

> ⚠️ **주의**: 이 API는 테스트 용도로만 사용해야 합니다.

**Path Parameters**
| 필드 | 타입 | 설명 |
|------|------|------|
| currentRatingPoint | Integer | 현재 레이팅 점수 |
| gameScore | Integer | 게임 점수 |

**Response**
```json
{
  "isSuccess": true,
  "code": "2000",
  "message": "OK",
  "data": 25
}
```

**응답 값**
- 레이팅 점수 변화량 (양수: 상승, 음수: 하락)

**참고사항**
- 이 API는 레이팅 시스템의 동작을 테스트하는 용도입니다.
- 실제 게임에서는 게임 종료 API를 통해 자동으로 레이팅이 계산됩니다.

## 랭크 시스템 설명

### 랭크 티어
| 티어 | 레이팅 범위 | 색상 |
|------|------------|------|
| BRONZE | 0 ~ 999 | 🟤 브론즈 |
| SILVER | 1000 ~ 1499 | ⚪ 실버 |
| GOLD | 1500 ~ 1999 | 🟡 골드 |
| PLATINUM | 2000 ~ 2499 | 💚 플래티넘 |
| DIAMOND | 2500 ~ 2999 | 💎 다이아몬드 |
| MASTER | 3000 ~ 3499 | 🔴 마스터 |
| GRANDMASTER | 3500 ~ 3999 | 🟣 그랜드마스터 |
| CHALLENGER | 4000+ | 🌟 챌린저 |

### 랭크 레벨
각 티어는 5개의 레벨로 구분됩니다:
- `LEVEL_5`: 최하위 레벨
- `LEVEL_4`
- `LEVEL_3`
- `LEVEL_2`
- `LEVEL_1`: 최상위 레벨 (다음 티어 직전)

### 레이팅 계산 방식
- 게임 점수가 높을수록 레이팅이 많이 상승합니다.
- 현재 레이팅이 높을수록 얻는 레이팅 점수가 감소합니다 (상위권 경쟁 심화).
- 점수가 낮으면 레이팅이 하락할 수 있습니다.
